Nozaki Components | 野崎コンポーネント Filter the components with the input field.
Component Name Description
nozaki-header A page or section header with an icon/logo/image.
nozaki-footer A footer which allows any content to be used.
nozaki-input A fancy text input box with placeholder support.
nozaki-button Buttons, square, rectangle and rounded corners, with hover and down states.
nozaki-table A handsome table with hover states and alternating row colors.
nozaki-list A pretty list with hover states for each item.
nozaki-table-filter A really fancy table with a title and subtitle with input box to search or filter the contents.
